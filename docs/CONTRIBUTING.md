'''# Contributing to Logos_Agency

We welcome contributions from the community! By participating in this project, you agree to abide by our Code of Conduct.

## 1. Getting Started

1.  **Fork** the repository on GitHub.
2.  **Clone** your forked repository locally:
    ```bash
    git clone https://github.com/YOUR_USERNAME/Logos_Agency.git
    cd Logos_Agency
    ```
3.  **Create a new branch** for your feature or fix:
    ```bash
    git checkout -b feature/my-new-feature
    ```

## 2. Making Changes

*   **Code Style:** Please adhere to the existing code style.
*   **Testing:** All new features or bug fixes must be accompanied by appropriate tests.
*   **Documentation:** Update the relevant documentation in the `/docs` folder for any changes to the architecture, roadmap, or performance.

## 3. Submitting a Pull Request (PR)

1.  **Commit** your changes with a clear and descriptive commit message.
2.  **Push** your branch to your fork on GitHub.
3.  **Open a Pull Request** against the `main` branch of the `Triune-Oracle/Logos_Agency` repository.
4.  **Fill out the Pull Request Template** (which will automatically populate).
5.  Ensure all **CI checks pass** (green checks).

## 4. Code of Conduct

Please note that this project is released with a Contributor Code of Conduct. By participating in this project, you agree to abide by its terms.

## 5. Need Help?

If you have any questions or need clarification, please open an [Issue](../../issues) before starting work.
